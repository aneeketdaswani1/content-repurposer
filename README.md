# AI Content Repurposer Pro

An AI-powered React application that analyzes long-form text and automatically repurposes it into platform-specific formats with customizable tones.

## Features
* **NLP Analysis:** Extracts readability scores, emotional sentiment, and keywords.
* **Tone Transformation:** Adapts content to Professional, Casual, Humorous, and more.
* **Multi-Platform Output:** Formats for 𝕏, LinkedIn, Instagram, YouTube, and Podcasts.

## Tech Stack
* React + Vite
* Anthropic Claude API 

## Local Development
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Create a `.env` file at the root and add your API key: `VITE_ANTHROPIC_API_KEY=your_key_here`
4. Run `npm run dev` to start the local development server.
